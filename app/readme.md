# Deployment App

Folder ini berisi kode untuk aplikasi dashboard interaktif (Demo).

## Daftar File
- **`app.py`**: Script utama aplikasi berbasis **Streamlit**.
  - Memuat model yang sudah dilatih dari `outputs/models/`.
  - Menyediakan antarmuka untuk memilih tanggal dan melihat prediksi harga Bitcoin.
  - Menampilkan visualisasi interaktif menggunakan Plotly.
