# Configuration

Folder ini berisi file konfigurasi untuk mengatur parameter eksperimen tanpa perlu mengubah kode program.

## Daftar File
- **`config.json`**: File konfigurasi utama.
  - **Data Params**: Sequence length (60), path file data.
  - **Training Params**: Learning rate, Batch size, Epochs.
  - **Paths**: Lokasi penyimpanan model dan log.
