# Data Folder

Folder ini menyimpan semua dataset yang digunakan dalam proyek prediksi harga Bitcoin.

## Struktur

- **raw/**: Menyimpan data mentah (`btc_1h_data_2018_to_2025.csv`). Data ini adalah data historis per jam yang belum diolah.
- **processed/**: Menyimpan data yang sudah dibersihkan dan diproses (`btc_preprocessed.csv`).
